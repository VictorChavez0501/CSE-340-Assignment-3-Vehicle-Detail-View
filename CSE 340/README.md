# Assignment 3 - Inventory Detail View (PostgreSQL)

Instrucciones rápidas:

1. Copia este proyecto a tu máquina.
2. `npm install`
3. Crear la base de datos en PostgreSQL y correr `db/init_postgres.sql` en psql o PgAdmin.
4. Crear un archivo `.env` con las variables según `.env.example`.
5. `npm start` y abrir `http://localhost:3000`

Variables .env:
PG_HOST, PG_PORT, PG_USER, PG_PASSWORD, PG_DATABASE, NODE_ENV, PORT
